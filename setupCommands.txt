RIZAL EXPLORER STARTUP COMMANDS

---------------------------------------------------------
OPTION 1: BAGONG STARTUP (FIRST-TIME SETUP)
---------------------------------------------------------
Use this if you just cloned the project or need to reset everything.

1. START INFRASTRUCTURE
   docker-compose up -d

2. BACKEND INITIALIZATION
   cd backend
   poetry install
   
   # --- MODEL ---
   # The custom "rizal-xlm-r-dapt" model is ALREADY PRE-TRAINED and bundled at:
   #   backend/app/models/rizal-xlm-r-dapt
   # NO TRAINING NEEDED on first setup. Skip directly to the migration steps below.
   #
   # (OPTIONAL) Only re-train if you want to update the model with new corpus data:
   #   1. poetry add "accelerate>=0.26.0"   ← install training dependency first
   #   2. poetry run python scripts/train_dapt.py
   #   3. Move model_output/rizal-xlm-r  →  app/models/rizal-xlm-r-dapt
   #   4. Re-run seed_db.py and seed_full_db.py to re-index the database
   # ----------------------
   
   poetry run python scripts/migrate_source_type.py
   poetry run python scripts/migrate_is_short.py
   poetry run python scripts/migrate_original_index.py
   poetry run python scripts/migrate_dapt_column.py
   poetry run python scripts/character_index.py
   
   # --- SEEDING ---
   # 1. Search Bar Embeddings (Base XLM)
   poetry run python scripts/seed_db.py
   poetry run python scripts/seed_full_db.py
   
   # 2. Sanggunian Embeddings (DAPT)
   poetry run python scripts/seed_dapt_db.py
   poetry run uvicorn app.main:app --reload

3. FRONTEND INITIALIZATION
   (Open a new terminal)
   cd frontend
   npm install
   npm run dev

---------------------------------------------------------
OPTION 2: KASUNOD NA STARTUP (SUBSEQUENT RUNS)
---------------------------------------------------------
Use this for daily development after the first setup is done.

1. START DOCKER (If not running)
   docker-compose up -d

2. RUN BACKEND
   cd backend
   poetry run uvicorn app.main:app --reload

3. RUN FRONTEND
   (Open a new terminal)
   cd frontend
   npm run dev

🚀 Access the app at: http://localhost:3000
