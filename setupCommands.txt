RIZAL EXPLORER - SETUP COMMANDS

1️⃣ FIRST-TIME STARTUP
--------------------
docker-compose up -d

# Backend
cd backend
poetry install
poetry run python scripts/seed_db.py
poetry run python scripts/seed_full_db.py
poetry run python scripts/migrate_source_type.py
poetry run uvicorn app.main:app --reload

# Frontend (New Terminal)
cd frontend
npm install
npm run dev


2️⃣ SUBSEQUENT STARTUP
--------------------
docker-compose up -d

# Backend
cd backend
poetry run uvicorn app.main:app --reload

# Frontend (New Terminal)
cd frontend
npm run dev

🚀 URL: http://localhost:3000
